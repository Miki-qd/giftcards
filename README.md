# Gift Cards Management System

A secure and lightweight web application built using **Laravel**, **Inertia.js**, and **Vue 3** (with Tailwind CSS and shadcn-vue) to manage gift cards and track operations.

## Key Features
- **Dashboard**: Overview of system statistics (active/used cards, recent actions).
- **Gift Cards Management**: Full CRUD operations for gift cards (Create, Read, Update, Delete) and CSV export.
- **Activity Log (History)**: Track system actions (creates, updates, deletions) with details on who did what and when, including CSV export.
- **Secure Authentication**:
  - Registration requires an **Invite Token** generated by the administrator (`admin@example.com`).
  - Support for **Two-Factor Authentication (2FA)** and **Passkeys (WebAuthn)**.
  - Automatic session timeout due to inactivity (5 minutes).

## Requirements
- PHP 8.2+
- Composer
- Node.js & NPM

## Quick Start / Setup

1. **Install Backend Dependencies**:
   ```bash
   composer install
   ```

2. **Install Frontend Dependencies**:
   ```bash
   npm install
   ```

3. **Environment Configuration**:
   Copy the example environment file and configure your database. By default, it is configured to use **SQLite** (no database server installation required):
   ```bash
   copy .env.example .env
   ```
   *Note: Ensure `DB_CONNECTION=sqlite` is set in your `.env`.*

4. **Initialize Database**:
   Create the SQLite database file and run the migrations & seeders:
   ```bash
   # Create database file
   copy NUL database\database.sqlite
   
   # Run migrations and seed data
   php artisan migrate --seed
   ```

5. **Start the Application**:
   Run the Laravel local development server:
   ```bash
   php artisan serve
   ```
   And run the Vite development server in another terminal:
   ```bash
   npm run dev
   ```

6. **Access the App**:
   Open `http://127.0.0.1:8000` in your browser.
   - **Default Admin Account**:
     - **Email**: `admin@example.com`
     - **Password**: `password` (or as configured in seeders)
     - Use the admin account to generate Invite Tokens for registering new users.
